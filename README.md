# 🎯the-shape-of-voice|手语翻译系统项目
## 📝项目简介
本项目是我们6人软件工程团队打造的双向手语翻译工具，专为打破听障群体与健听人士的沟通壁垒设计。项目依托视觉识别算法，实现手语与文字/语音实时双向互译、场景化精准翻译，以无门槛、轻量化方式完成无障碍实时沟通，以公益为导向，希望用轻量化技术搭建平等顺畅的沟通桥梁，助力无障碍社会建设。

## ✨核心功能模块及亮点
| 功能模块 | 描述 |
| --- | --- |
| 双向手语翻译 | 完成手语、文字、语音之间双向转换交流 |
| 场景化专属词库 | 匹配就医日常出行场景，提升翻译准确度 |
| 网页端视觉识别 | 依托网页捕捉手势动作并智能解析判定 |

| 创新点 | 描述 |
| --- | --- |
| 双向交互 | 市面同类产品多为“手语→文字/语音”单向输出，本项目实现双向实时互译，形成完整沟通闭环 |
| 场景化适配 | 针对就医问诊、日常出行等常见场景定制词库，解决通用产品翻译效率低，更贴合用户真实痛点 |
