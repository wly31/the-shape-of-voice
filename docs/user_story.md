# 👥 用户故事

| 用户角色 | 用户故事 | 核心价值 |
| :--- | :--- | :--- |
| <span style="white-space: nowrap;">听障人士</span> | 我想要打开摄像头对准自己的手语动作，系统能够实时将手语动作转换为文字并进行语音播报；同时可以查看历史识别记录 | 实现听障人士与健听人士的无障碍实时沟通，通过历史记录回顾沟通内容，避免信息遗漏 |
| <span style="white-space: nowrap;">健听人士</span> | 我想要输入文字或语音，系统可以实时生成标准手语动画，同时支持保存生成的手语动画文件 | 让健听人群能够主动向听障人士传递信息，支持内容保存与转发，方便重复查看与沟通复用 |
| <span style="white-space: nowrap;">听障学生</span> | 我想要将课堂语音转为文字、通过手语和师生交流，对照标准手势练习手语；搭配简洁护眼大字体界面 | 适配校园教学场景，帮助听障学生跟上课堂进度、参与互动练习，弱网环境也可正常使用，提升学习体验与舒适度 |
| <span style="white-space: nowrap;">手语翻译员</span> | 我想要实现手语、语音、文字双向快速转换，支持自定义高频词库与短语，可查询、回放标准手语例句与动作 | 辅助翻译人员日常练习，提升现场翻译速度、准确率与工作效率，满足专业翻译工作需求 |
| <span style="white-space: nowrap;">医护人员</span> | 我想要通过摄像头识别患者问诊手语，实时转文字并语音播报；同时可录入口述问诊内容，一键生成标准手语动画供患者查看 | 适配医院急诊问诊场景，快速完成医患双向沟通，高效获取患者病情信息，辅助医护人员快速做出医疗判断 |
| <span style="white-space: nowrap;">产品管理员</span> | 我想要在后台统计查看学校、医院、公共场所等不同场景的高频手语查询词汇数据 | 根据真实场景使用数据迭代优化手语词库，针对性提升各场景下的翻译准确率，优化产品整体性能 |
