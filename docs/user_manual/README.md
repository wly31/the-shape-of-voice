# the shape of voice 手语双向翻译系统 - 用户使用手册

## 欢迎使用 the shape of voice！

SignTranslate 是一款实时手语双向翻译系统，能够实现手语与文字/语音之间的双向转换：

- **手语转文字/语音**：支持实时摄像头、图片、视频三种输入方式，识别手语手势（A-Z 字母）并输出文字和语音
- **文字/语音转手语**：支持文字输入和语音输入，转换为对应的3D手语动画

系统旨在为听障人士与健听人士搭建便捷的沟通桥梁。

## 快速开始

- Windows 10/11 或 macOS

- Python 3.10 或更高版本
