# 安装与运行指南

## 系统要求

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| 操作系统 | Windows 10 / macOS 11 | Windows 11 / macOS 13 |
| 内存 | 4GB | 8GB 或以上 |
| 摄像头 | 720p | 1080p |
| 浏览器 | Chrome 90+ / Edge 90+ | Chrome 最新版 |

## 下载与安装

### 方式一：直接运行（推荐）

1. 下载项目压缩包并解压
2. 进入 `SignTranslate` 文件夹
3. 双击运行 `start.bat`（Windows）或 `start.sh`（macOS）
4. 浏览器将自动打开系统界面

### 方式二：手动启动

1. 打开终端（命令提示符或 PowerShell）

2. 进入项目目录：
```bash
cd SignTranslate
