# 使用指南

## 功能概览

SignTranslate 提供两大核心功能：

| 模式 | 功能说明 |
|------|----------|
| 手语转文字，语音 | 识别实时手语手势、手语图片或视频，输出文字和语音播报 |
| 语音，文字转手语 | 输入文字或语音，输出3D手语动画 |

## 模式一：手语转文字，语音

### 三种输入方式

#### 1. 实时摄像头识别

**操作步骤**：

1. 点击「手语转文字」→「实时识别」
2. 允许浏览器摄像头权限
3. 将手部置于摄像头前，保持手势稳定
4. 系统自动识别并显示结果

**界面布局**：

| 摄像头预览 | 识别结果 |
|------------|----------|
| 👋 手部画面 | **A** (置信度 95%) |
|            | [🔊 语音播报] 按钮 |

| 控制按钮 |
|----------|
| [🟢 开始] [🔴 停止] [📷 切换摄像头] |

#### 2. 上传图片识别

**操作步骤**：

1. 点击「手语转文字」→「上传图片」
2. 点击上传区域或拖拽图片到指定区域
3. 系统自动识别图片中的手语手势
4. 显示识别结果并语音播报

**界面布局**：

| 上传区域 | 识别结果 |
|----------|----------|
| 📁 点击或拖拽图片上传 | **你好** (置信度 92%) |
| 支持格式：JPG、PNG、BMP、WEBP | [🔊 语音播报] 按钮 |

| 操作按钮 |
|----------|
| [📤 上传] [🔄 重新识别] [💾 导出结果] |

#### 3. 上传视频识别

**操作步骤**：

1. 点击「手语转文字」→「上传视频」
2. 选择视频文件上传
3. 系统逐帧分析，识别连续手语
4. 输出完整句子翻译并语音播报

**界面布局**：

| 视频播放区 | 识别结果 |
|------------|----------|
| 🎬 视频预览画面 | **今天天气很好** (置信度 89%) |
| 进度条 [====] 00:05 / 00:15 | [🔊 语音播报] 按钮 |

| 操作按钮 | 导出选项 |
|----------|----------|
| [📤 上传视频] [▶️ 播放] [⏸️ 暂停] | [📄 导出 TXT] [📊 导出 CSV] [🎬 导出 SRT] |

**支持格式**：MP4、AVI、MOV、WEBM

**建议时长**：不超过 5 分钟

**输出说明**：

| 输出类型 | 说明 |
|----------|------|
| 文字显示 | 识别出的完整句子显示在屏幕中央 |
| 语音播报 | 自动朗读识别结果，可手动开关 |
| 字幕导出 | 支持导出 SRT 格式字幕文件 |

## 模式二：语音，文字转手语

## 模式二：文字/语音转手语

### 两种输入方式

#### 1. 文字输入

**操作步骤**：

1. 点击「文字/语音转手语」→「文字输入」
2. 在输入框中输入文字
3. 点击「生成手语」
4. 系统播放 3D 手语动画

**界面布局**：

| 文字输入区 | 动画展示区 |
|------------|------------|
| ┌─────────────────────────────────┐ |            |
| │ 你好，今天天气很好               │ |   🧑 3D手语动画 |
| └─────────────────────────────────┘ |            |
| [生成手语] 按钮 |            |

| 手势分解 | 动画控制 |
|----------|----------|
| [你] [好] [今天] [天气] [好] | [▶️ 播放] [⏸️ 暂停] [🔁 循环] |

| 显示模式 | 速度调节 |
|----------|----------|
| [👤 3D模型] [🖐️ 仅手部] [📖 分步演示] | [0.5x] [1.0x] [1.5x] |

#### 2. 语音输入

**操作步骤**：

1. 点击「文字/语音转手语」→「语音输入」
2. 点击麦克风按钮 🎤
3. 对着麦克风说话
4. 系统自动将语音转为文字，再生成手语动画

**界面布局**：

| 语音输入区 | 识别文字 |
|------------|----------|
| 🎤 点击麦克风开始说话 | **你好，今天天气很好** |
| 支持语言：中文普通话 | [确认] [重新输入] |

| 动画展示区 |
|------------|
| 🧑 3D手语动画 |

| 手势分解 | 动画控制 |
|----------|----------|
| [你] [好] [今天] [天气] [好] | [▶️ 播放] [⏸️ 暂停] [🔁 循环] |

| 显示模式 | 速度调节 |
|----------|----------|
| [👤 3D模型] [🖐️ 仅手部] [📖 分步演示] | [0.5x] [1.0x] [1.5x] |

### 输出说明

| 输出类型 | 说明 |
|----------|------|
| 3D 动画 | 完整展示手语动作 |
| 手势分解 | 逐词显示对应的手势名称 |
| 文字同步 | 动画播放时同步显示当前手势对应的文字 |
