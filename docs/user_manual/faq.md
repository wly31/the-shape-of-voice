#  常见问题 (FAQ)

## 1. 系统支持哪些平台？

支持 Windows 10/11、macOS 11+。

## 2. 摄像头无法打开怎么办？

**解决方法**：
- 检查浏览器地址栏是否允许摄像头权限
- 关闭其他占用摄像头的应用（如微信、Zoom）
- 重启浏览器或电脑

## 3. 手语识别不准确怎么办？

**解决方法**：

| 问题 | 解决方案 |
|------|----------|
| 光线不足 | 增加光源，避免逆光 |
| 手势不规范 | 参考手势图调整手势角度 |
| 移动太快 | 每个手势保持 0.5-1 秒 |
| 背景杂乱 | 换到简洁背景（如白墙） |

## 4. 提示“未检测到手部”怎么办？

**解决方法**：
- 确保手部完整在画面中央
- 调整光线，避免逆光
- 保持手势稳定
- 背景尽量简洁

## 5. 语音播报没有声音怎么办？

**解决方法**：
- 检查电脑音量是否开启
- 检查浏览器是否允许自动播放
- 点击语音播报按钮手动触发

## 6. 某些文字没有对应的手语动画怎么办？

当前支持常用词汇（约 500 个）。对于未收录的词汇，系统会按单字拆分显示，也可使用「自定义词汇」功能添加。

## 7. 识别结果如何导出？

| 功能 | 导出格式 |
|------|----------|
| 图片识别 | TXT、CSV |
| 视频识别 | TXT、CSV、SRT |
| 历史记录 | TXT、CSV |
